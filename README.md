# MusicStudio

**MusicStudio** is a digital audio workstation.

![MusicStudio](Res/MusicStudio.png)

![Piano](Res/Piano.png)

## Build instructions

### Prerequisites

#### macOS

    xcode-select install

### Setup:

    eval $(./bootstrap)

### Build:

    ninja -C build

### Run:

    ./build/bin/music-studio
